# WDACISCO – Academia de Redes y Ciberseguridad 🚀

¡Bienvenido a la plataforma líder para la preparación del CCNA! **WDACISCO** es una academia técnica interactiva diseñada para estudiantes de redes que buscan dominar el ecosistema Cisco desde cero.

## ✨ Características Principales
- **12 Módulos Académicos:** Cubriendo desde el Modelo OSI hasta Automatización de Redes.
- **Simulador CLI Integrado:** Practica comandos de Cisco IOS directamente en el navegador.
- **Profesor IA (AI Tutor):** Un asistente inteligente especializado en redes para resolver tus dudas en tiempo real.
- **Motor de Evaluación:** Tests de 10 preguntas por módulo con retroalimentación inmediata.
- **Arquitectura Estática:** Optimizada para un despliegue rápido en GitHub Pages.

## 🛠️ Tecnologías Utilizadas
- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript (ES6+).
- **Simulación:** Motor de comandos basado en lógica condicional JS.
- **IA:** Integración de base de conocimientos técnica (Knowledge Base).

## 📂 Estructura del Proyecto
- `/modules`: Contenido teórico-práctico detallado.
- `/labs`: Escenarios de laboratorios guiados.
- `/tests`: Motor de exámenes y validación de conocimientos.
- `/ai`: Lógica del Tutor IA.

## 🚀 Despliegue
Este proyecto está listo para ser servido mediante **GitHub Pages**. Solo clona el repo y activa la opción en los ajustes de tu repositorio.
